Common CSS properties

Property Description Syntax
Background-image sets the background image of an element background-image: url("moon.jpeg")
Font Family used to specify the typeface in a rule set font-family: Verdana; font-family: "Courier New";
foreground text color foreground text color of an element can be set color : green ;
Background Color controls the background color of elements. background-color: blue;
Text Align used to set the text alignment of inline contents: left , right , or center text-align: right;
Font Weight used to set the weight (boldness) of text font-weight: bold;
Font Size used to set text sizes. Font size values font-size: 30px;
Opacity Can be used to control the transparency of an element. opacity: 0.5;
